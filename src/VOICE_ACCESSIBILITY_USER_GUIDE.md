# Voice-Guided Accessibility Feature - User Guide

## Overview
The Indian National e-Voting Portal now includes a comprehensive **Voice-Guided Accessibility System** designed specifically for blind and visually impaired users. This feature enables users to complete the entire voting process using only voice commands and spoken input, with step-by-step audio guidance throughout.

## Key Features

### üé§ Complete Hands-Free Experience
- **Automatic voice guidance** at every step
- **Automatic listening** after each instruction
- **Automatic field filling** based on voice input
- **Automatic progression** through the registration flow

### üîä What the System Does Automatically

1. **Speaks instructions clearly** for each field
2. **Listens for your voice input** without needing to click anything
3. **Fills form fields automatically** as you speak
4. **Moves to the next field** when complete
5. **Provides progress updates** (e.g., "5 digits entered, 7 more needed")
6. **Confirms your selections** before submission

## How to Enable Voice Mode

### On Any Page:
1. Look for the **blue "Voice Accessibility" notice** at the top of the page
2. Click **"Enable Voice Mode"** button, OR
3. Click the **floating microphone button** (bottom-right corner with rainbow gradient)

### First-Time Setup:
- Your browser will ask for **microphone permission**
- Click **"Allow"** to enable voice features
- The system will immediately start guiding you

## Global Voice Commands

These commands work on **any page** in voice mode:

| Command | Action |
|---------|--------|
| **"Repeat"** or **"Again"** | Repeat the last instruction |
| **"Cancel"** or **"Exit"** | Exit voice mode |
| **"Back"** or **"Previous"** | Go to previous page |

## Tips for Best Experience

### üéØ Speaking Tips:
1. **Speak clearly** but naturally
2. **Pause briefly** between digits for numbers
3. **Wait for the beep** before speaking (indicates listening started)
4. **Listen completely** to instructions before responding

### üîä Audio Quality:
- Use a **quiet environment** if possible
- Speak at **normal volume** (not too loud or soft)
- **Position microphone** 6-12 inches from your mouth
- Use **headphones** to hear instructions clearly

### üîÑ If Something Goes Wrong:
- Say **"Repeat"** to hear the instruction again
- Say **"Clear"** or **"Reset"** to start field over
- Say **"Back"** to return to previous page
- Say **"Cancel"** to exit voice mode entirely

## Troubleshooting

### Problem: "Microphone permission denied"
**Solution:** 
1. Click the üîí lock icon in browser address bar
2. Find "Microphone" permission
3. Change to "Allow"
4. Refresh the page

### Problem: "Voice recognition not supported"
**Solution:**
- Use Google Chrome (best compatibility)
- Update your browser to latest version
- Try on a different device

### Problem: System not listening
**Solution:**
- Say **"Repeat"** to restart current instruction
- Check if microphone icon shows **üé§ Listening**
- Speak after you hear the beep sound
- Check browser console for errors

### Problem: Wrong digits/text captured
**Solution:**
- Speak more slowly and clearly
- Say digits individually: "One. Two. Three." (not "one-two-three")
- Say **"Clear"** to reset the field
- Reduce background noise

### Problem: System speaks too fast/slow
**Note:** Currently set to 0.85x speed for accessibility. This can be adjusted in future updates based on user feedback.

## Technical Details

### Voice Recognition:
- Uses **Web Speech API** (SpeechRecognition)
- Language: **English (India)** - en-IN
- Mode: Single utterance per field
- Continuous listening with automatic restarts

### Text-to-Speech:
- Uses **Web Speech API** (SpeechSynthesis)  
- Language: **English (India)** - en-IN
- Rate: 0.85 (slower for clarity)
- Volume: 100%
